# AURORA Atelier (Next.js)

Premium multi-page photography website built with Next.js + TypeScript (App Router).

## Stack
- Next.js 15
- TypeScript
- App Router
- CSS design system (custom, non-template)

## Run
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Pages
- `/` home
- `/portfolio`
- `/services`
- `/pricing`
- `/process`
- `/systems`
- `/team`
- `/contact`
- `/cases/[slug]`

## Structure
- `/src/app/*` routes
- `/src/components/*` shared UI shell
- `/src/lib/site-content.ts` content model and reusable data
- `/src/app/globals.css` visual language and responsive rules
