# AURORA STUDIO System Playbook

## 1. 營運目標
- 24 小時內完成新詢價初次接觸
- 詢價 -> 預約轉換率 >= 35%
- 報價 -> 簽約轉換率 >= 45%
- 專案準時交件率 >= 95%

## 2. Pipeline 階段 (CRM)
1. New Lead: 新詢價進站
2. Qualified: 條件符合，進入諮詢
3. Proposal Sent: 已寄報價/提案
4. Negotiation: 方案確認與議價
5. Won: 已簽約與收訂
6. Fulfillment: 拍攝/後期中
7. Delivered: 完成交件
8. Lost: 未成交（記錄原因）

## 3. Lead 必填欄位
- 聯絡人姓名
- 公司/品牌（若有）
- Email / 電話 / LINE
- 拍攝類型（婚禮/品牌/人像/活動）
- 期望拍攝日期
- 預算區間
- 使用場景（官網/社群/廣告/新聞稿）
- 來源渠道（SEO/IG/介紹/廣告）

## 4. 標準作業時間 (SLA)
- 詢價自動回覆: 即時
- 人工首次回覆: 4 小時內
- 初步提案: 24 小時內
- 確認檔期與簽約: 48 小時內
- 交件: 依合約（預設 7-14 天）

## 5. 自動化規則 (Make/Zapier)
- 規則 A: 表單送出 -> 建立 CRM Lead -> 發送通知到 Email/Slack
- 規則 B: CRM 階段改為 Qualified -> 發送 Calendly 預約連結
- 規則 C: 階段改為 Won -> 發送訂金付款連結與拍攝前問卷
- 規則 D: 階段改為 Delivered -> 發送評價與轉介紹邀請

## 6. 合約與付款
- 訂金: 40% (簽約後 3 日內)
- 尾款: 60% (拍攝日前 3 日)
- 變更檔期: 依合約條款（需書面）
- 急件快修加價: 20%-35%

## 7. 交件規格
- 精修檔: JPG (sRGB, 長邊 4000px)
- 社群檔: JPG (1080 / 1350 / 1920)
- 備份保存: 6 個月
- 交件平台: Pixieset / 雲端資料夾

## 8. 每週營運檢討
- 本週新增詢價數
- 詢價 -> 預約轉換率
- 報價 -> 簽約轉換率
- 平均客單價
- 交件滿意度/NPS
- 流失原因 Top 3
