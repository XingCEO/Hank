# Professional Photography Website

## Quick Start
1. Open `index.html` directly in browser, or
2. Run a local server:
   - `cd /Users/xuser/Documents/攝影網站`
   - `python3 -m http.server 8080`
   - open `http://localhost:8080`

## Site Pages
- `index.html` Home
- `portfolio.html` Portfolio
- `services.html` Services
- `pricing.html` Pricing
- `process.html` Workflow SOP
- `systems.html` Operations System
- `team.html` Team
- `contact.html` Booking
- `case-wedding.html` Case Study (Wedding)
- `case-brand.html` Case Study (Brand)

## Customize Before Go-Live
- Replace brand name and contacts in all pages.
- Replace Unsplash placeholders with your own photos.
- Connect contact form to your backend/automation tool.
- Add GA4 and Search Console tags.
- Update legal pages (privacy/terms) if needed.

## Core Files
- `styles/main.css` visual system and responsive layout
- `scripts/main.js` mobile menu + reveal animation
- `SYSTEM_PLAYBOOK.md` CRM/SOP/automation checklist
